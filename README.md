# DIGM_FINAL_PROJECT
I want to make a scene generator. It would take basic geometry and arrange them in ways that allude to a forest. The scene would have a cabin, rocks, trees, as well as a cloud and a sun. You would be able to rearrange this scene however you want, as well as be similar to a terrain/ landscape scatter/generator.

The Planned functions:
create_hero_asset: For the cabin and specific props
create_tree: To handle your 3 variations
create_environment_element: For the clouds/sun
scatter_collection: The function that will arrange the forest and allow for changes and such
